# Projeto H4B FootBall Club

Este projeto possui um frontend em HTML/CSS/JS e um backend em Node.js.

## Como executar

1. Abra um terminal.
2. Navegue até a pasta do backend:
   ----------
   cd backend
   ----------
3. Inicie a API:
   ---------- 
   npm start
   ---------- 
4. Abra o frontend pelo arquivo `home.html` no navegador.

## Observações

- Se for a primeira vez que estiver executando o projeto, pode ser necessário instalar dependências dentro de `backend`:

  baixar node.js pelo navegador

  clone o repostório

  npm install

- O backend usa `server.js` como arquivo de entrada pelo script `start`.
